# Computer Vision Lab: Practical Implementations

A hands-on collection of computer vision algorithms and techniques implemented in Python.

## 🎯 Core Techniques

### Image Processing
- **Basic Operations**: Reading, writing, color conversions, resizing
- **Point Processing**: Contrast stretching, histogram equalization, gamma correction
- **Intensity Transformations**: Negative, log, power-law

### Feature Extraction
- **Edge Detection**: Sobel, Prewitt, Canny
- **Texture Analysis**: Gabor filters (multi-orientation), LBP
- **Statistical Features**: Autocorrelation, histogram analysis

### Object Detection
- **Traditional**: Haar Cascades, Template Matching, HOG
- **Deep Learning**: YOLOv8, MobileNet SSD

### Pattern Recognition
- **Preprocessing**: Auto-scaling (Z-score), normalization
- **Dimensionality Reduction**: PCA
- **Clustering**: K-Means
- **Classification**: ANN, CNN

### Advanced Topics
- **CBIR**: Color histogram & texture-based image retrieval
- **Optimization**: Particle Swarm Optimization (PSO)
- **Biometrics**: Face detection (Haar) & recognition (LBPH)
