**Motion Detection \& Intrusion Alert System**

**Description**

This project implements a real-time motion detection system using OpenCV. It detects movement via frame differencing and logs intrusion events.



**Features**

Real-time motion detection

Frame differencing technique

Contour detection

Motion alert display

Event logging with timestamp

Text-based log storage

Technologies Used

Python

OpenCV

datetime module

How to Run

pip install opencv-python python app.py



**Output**

Webcam window with bounding box

motion\_log.txt file storing alerts
