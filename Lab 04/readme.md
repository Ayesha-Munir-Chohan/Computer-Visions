**Computer Vision Image Enhancement Project**

This project demonstrates classical image enhancement algorithms implemented in Python using OpenCV.



The goal is to improve image visibility in difficult environments such as:



Low lighting

Fog or smog

Underwater environments

Thermal satellite imagery

Implemented Algorithms

Contrast Stretching

Gamma Correction

Histogram Equalization

CLAHE (Adaptive Histogram Equalization)

These techniques improve image contrast, brightness, and visibility of important features.



**Project Tasks**

Task 1 – Contrast Stretching

Enhancing low contrast grayscale images by expanding intensity range.



Task 2 – Gamma Correction

Adjusting brightness levels to reveal hidden details in dark images.



Task 3 – Histogram Equalization

Improving contrast in medical and thermal images.



Task 4 – CLAHE Enhancement

Adaptive contrast enhancement useful for nighttime driving scenes.



Task 5 – Combined Enhancement

Applying multiple enhancement techniques to improve foggy images.
