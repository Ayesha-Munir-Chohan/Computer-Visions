\# Face Detection \& Recognition Project



\## 📌 Description

This project implements multiple real-world face recognition systems using OpenCV and LBPH algorithm.  

All implementations use \*\*datasets and videos instead of webcam\*\*.



\---



\## 🧩 Implemented Systems



1\. Smart Attendance System  

2\. Access Control System  

3\. Celebrity Recognition  

4\. Intruder Detection  

5\. Criminal Identification  



\---



\## ⚙️ Features

\- Face detection using Haar Cascades

\- Face recognition using LBPH

\- Video-based processing

\- CSV logging

\- Real-time predictions



\---



\## ⚡ Performance Optimizations

\- Frame skipping

\- Image resizing

\- Grayscale processing



\---



\## 📁 Requirements

\- Python 3.13

\- OpenCV (contrib version)

\- NumPy

\- Pandas

\- Matplotlib
